Overview

課程概要

專為完全零基礎的繁體中文使用者設計，透過清晰的學習路徑與語音輔助，打好日語基礎。

🎯

學習目標

8 週內掌握平假名・片假名・基本文法・日常會話，達到 JLPT N5 程度

🗣️

語音教學

每課內建 Web Speech API 發音按鈕，點擊即可聽正確發音，隨時練習

📝

繁體中文輔助

全部解說以繁體中文呈現，對照注音符號與羅馬拼音，降低學習門檻

8-Week Curriculum

八週課程表

每週一課，循序漸進，從發音基礎到日常會話

01 第 1 週

日語概論 · 發音基礎

假名起源與結構 · 母音（あいうえお）· 子音 · 日語聲調入門 · 發音器官與口型

02 第 2 週

平假名（上）

あ行～な行（清音 25 字）· 筆順與書寫 · 單字練習 · 聽音辨字

03 第 3 週

平假名（下）

は行～わ行・ん · 濁音（がぎぐげご）· 半濁音（ぱぴぷぺぽ）· 拗音（きゃ・きゅ・きょ）

04 第 4 週

片假名 · 外來語

ア行～ワ行・ン · 濁音・半濁音・拗音 · 長音與促音 · 外來語書寫規則

05 第 5 週

基本問候 · 自我介紹

です・ます 體 · 禮貌用語（おはよう・こんにちは等）· 數字 1～100 · 自己紹介文

06 第 6 週

動詞與形容詞入門

五段動詞與一段動詞 · い形容詞與な形容詞 · 現在式・否定形 · 日常用語例句

07 第 7 週

助詞與句型結構

助詞：は・が・を・に・で・へ · 疑問句・否定句・過去式 · 時間表達

08 第 8 週

日常會話 · 總複習

購物・餐廳・問路情境會話 · 文法總整理 · 簡短對話練習 · 成果發表

Teaching Method

教學方式

多重輔助工具，讓發音學習更直覺

🔊 語音輔助

每課提供 Web Speech API 即時發音按鈕，點選假名或單字即可聆聽標準日語發音。未來 App 將加入語速調整與慢速跟讀模式。

📖 繁體中文對照

所有文法解說、單字釋義、練習指示皆以繁體中文呈現。必要時附注音符號輔助發音記憶。

⭐ 每課附小測驗

每週課程結束後提供互動式小測驗（選擇題、聽力題、填空題），即時核對答案。App 將內建自動批改與分數記錄功能，搭配底部 tab 導航流暢切換。

  • 假名聽力辨識
  • 單字日中配對
  • 文法選擇題
  • 簡短造句
