# 日語入門課程

繁體中文初學者專用的日語入門互動課程，從零開始，8 週學會日常會話。

## 專案結構

```
├── README.md          # 本文件
├── docs/
│   ├── DESIGN.md      # 設計語言參考（Sentry 主題）
│   └── plan.html      # 課程整體計劃書（規劃階段文件）
└── app/
    └── index.html     # 互動式課程 App（單頁 HTML）
```

## 使用方式

### 課程 App

直接開啟 `app/index.html` 即可開始學習：

- **🏠 首頁** — 課程概覽與學習進度環
- **📖 課程** — 8 週課程列表，點擊進入各課內容
- **✍️ 測驗** — 從課程進入測驗，提交後自動批改
- **📊 進度** — 各課分數與完成率統計

### 發音功能

點擊假名表或單字旁的 ▶ 按鈕，即透過 Web Speech API 朗讀日語發音。

### 進度儲存

學習進度與測驗分數自動儲存於瀏覽器 localStorage，關閉頁面後不遺失。

## 課程大綱

| 週次 | 主題 |
|------|------|
| 第 1 週 | 日語概論 · 發音基礎（あいうえお） |
| 第 2 週 | 平假名（上）— あ行～な行 |
| 第 3 週 | 平假名（下）— 濁音・半濁音・拗音 |
| 第 4 週 | 片假名 · 外來語 |
| 第 5 週 | 基本問候 · 自我介紹 |
| 第 6 週 | 動詞與形容詞入門 |
| 第 7 週 | 助詞與句型結構 |
| 第 8 週 | 日常會話 · 總複習 |
